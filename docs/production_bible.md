# Production Bible

This document captures the core production rules for the show.

## Tone

- Light comedy with grounded character moments.
- Visuals should stay consistent with the configured style prefix.

## Characters

- Keep character voices consistent across episodes.
- Use the character data files as the source of truth.

## Locations

- Locations should be described with clear visual cues.
- Use location files for stable prompts.

## Quality checks

- Run `just check` before episode generation.
- Review output assets in `output/`.
- Regenerate any missing or low quality scenes.
